# odin-recipes

### Description
This website is my first project from **The Odin Project** curriculum. It is a simple recipe website that contains links to various recipes, demonstrating the use of anchors and list elements.

### Goals & Skills Demonstrated
By completing this project, I have practiced and strengthened my understanding of:
* Basic HTML structure (Boilerplate)
* Headings and paragraphs
* Unordered and ordered lists
* Absolute and relative links
* Working with the Linux terminal and Git
